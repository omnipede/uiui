# 01-emmet

Emmet 사용법, 단축키

https://docs.emmet.io/cheat-sheet/

# HTML 단축키

```text
lorem -> Dummy text 생성
lorem100 -> Dummy text 100 개 단어 생성.
div*5 -> div 5 개
.list-item#my-id -> list-item 클래스, my-id id div.
```

# CSS 단축키

```text
p20 -> padding: 20px
p20p -> padding: 20%
pt20p -> padding-top: 20%
pr100p -> padding-right: 100%
m20p -> margin: 20%
m:a -> margin: auto

pos -> position
posa -> position: absolute


fz60 -> font-size: 60px
fz10em -> font-size: 10em

fs -> font-style 
fsn -> font-style: normal

ta -> text-align
ta:c -> text-align: center
ta:l -> text-align: left
ta:r -> text-align: right

w -> width
w10 -> width: 10px
w10p -> width: 10%
w10em -> width: 10em
h10 -> height: 10px

lh -> line height
lh30 -> line height: 30 (주의, px 가 안 붙는다.)
```
